# Smart Schedule

> Smart instructor scheduling system using constraint optimization

## What it does
Automates university timetable generation — replacing manual
Excel-based scheduling with a constraint-aware engine that
handles availability, conflicts, and instructor priorities.

## Tech Stack
| Layer    | Technology        |
|----------|-------------------|
| Backend  | Python · FastAPI  |
| Database | PostgreSQL 15     |
| Frontend | React · Tailwind  |
| Auth     | JWT               |
